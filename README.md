## Hotel Price Checker 🏨

A Python-based tool built in Google Colab to track, compare, and visualise hotel prices across different cities and dates.

## 🚀 Features
- **Live Search**: (Planned) Integration with APIs like Amadeus or SerpApi for real-time data.
- **Data Analysis**: Uses Pandas to process and clean hotel pricing information.
- **Visualisation**: Generates bar charts and comparison graphs using Matplotlib and Seaborn.
- **Easy Configuration**: Simple setup for API keys and search parameters.
